Quick Flick
====

### Project Overview
Udacity Android Nanodegree - Popular Movies 2

This app will:

* Allow users to view and play trailers ( either in the youtube app or a web browser).
* Allow users to read reviews of a selected movie.
* Allow users to mark a movie as a favorite in the details view by tapping a button. This is for a local movies collection and does not require an API request.
